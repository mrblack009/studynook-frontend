## ABOUT

StudyNook is a premium library study room booking application designed to streamline student collaboration and quiet research. It provides a sleek, responsive, and glassmorphic user interface where users can search for active study rooms, filter workspaces by floor and amenities, manage their own listings as hosts, and schedule reservations securely. Developed as part of **Programming Hero Assignment 9**, this frontend focuses on robust context-based session management, Google single sign-on, and secure client-side routing.

## KEY FEATURES

1. **Premium Sticky Navigation & Theme Toggle:** A glassmorphism navigation header with dynamic theme settings supporting instant toggling between sleek light mode and premium dark mode.
2. **Interactive Listing Directory & Advanced Search Filters:** Real-time query listings utilizing search keywords, hourly rate sliders, floor levels, and individual comfort amenities (e.g., Whiteboards, Wi-Fi, Projectors).
3. **Interactive Host & Guest Panels:** Secure dashboards for hosts to post and manage study spaces, and for guests to review booking timelines and cancel active reservations dynamically.
4. **Google Authentication Integration:** Frictionless login experiences featuring secure Google Single Sign-On (SSO) alongside standard email credential login formats.
